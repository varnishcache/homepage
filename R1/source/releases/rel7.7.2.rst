Varnish Cache 7.7.2
===================

* Source download `varnish-7.7.2.tgz </downloads/varnish-7.7.2.tgz>`_

* SHA256=fda7750e1281a20e9cccc6c666481d2437fa7c0d816592c86e4fb7ba7fc464ad

Varnish Cache 7.7.2 is a security release that addresses the vulnerability
described in :ref:`VSV00017 <VSV00017>`.

If upgrading from 7.7.1, you should not expect to encounter any problems. However, VMODs have to be recompiled.

More information:

* List of most important `Changes in 7.7 <https://varnish-cache.org/docs/7.7/whats-new/changes-7.7.html>`_
* Help on `Upgrading to Varnish 7.7 <https://varnish-cache.org/docs/7.7/whats-new/upgrading-7.7.html>`_
* `Full changes.rst entry for 7.7.2 <https://github.com/varnishcache/varnish-cache/blob/7.7/doc/changes.rst#varnish-cache-772-2025-08-05>`_

For installation instructions including information about cloud images see
`the Varnish Installation Manual </docs/trunk/installation/index.html>`_
