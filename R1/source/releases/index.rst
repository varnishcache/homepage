Releases & Downloads
====================

Varnish Cache is released every 6 months.

.. list-table::
  :header-rows: 1

  * - Release
    - Date
    - EOL Date
    - Download
  * - :ref:`7.4.2 <rel7.4.2>`
    - 2023-11-13
    - 2024-09-15
    - `varnish-7.4.2.tgz </downloads/varnish-7.4.2.tgz>`_
  * - :ref:`7.3.1 <rel7.3.1>`
    - 2023-11-13
    - 2024-03-15
    - `varnish-7.3.1.tgz </downloads/varnish-7.3.1.tgz>`_
  * - :ref:`6.0.11 <rel6.0.11>`
    - 2022-11-08
    - Supported
    - `varnish-6.0.11.tgz </downloads/varnish-6.0.11.tgz>`_

All releases not mentioned above are End-Of-Life and unsupported.

For installation instructions including information about cloud images see
`the Varnish Installation Manual </docs/trunk/installation/index.html>`_
